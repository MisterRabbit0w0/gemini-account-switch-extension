A Chrome extension to open Gemini / AI Studio with a specific Google account in one click.

一个 Chrome 扩展：一键以指定 Google 账号打开 Gemini / AI Studio。

## Quick Start / 快速开始

1. Load `src/` as an unpacked extension in Chrome / 在 Chrome 中加载 `src/` 目录作为已解压的扩展
2. Add your Google accounts in Settings / 在设置中添加你的 Google 账号
3. Click the extension icon → pick an account → go! / 点击扩展图标 → 选择账号 → 出发！
